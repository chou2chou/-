# 面向听障学生的多模态教育智能体 - 后端

## 项目介绍
专为听障学生设计的智能教育辅助系统，提供实时字幕、知识点梳理、手语适配、课堂知识库、学情分析等功能，实现教育内容无障碍化。

## 技术栈
- FastAPI
- WebSocket 实时通信
- RAG 知识库检索
- MySQL 数据库
- Redis 缓存
- Docker 容器化部署

## 核心功能
1. 课堂实时字幕推送
2. 教学内容智能简化与知识点提取
3. PPT 自动构建知识库
4. 课堂学习情况分析与报告
5. 标准化接口 + 跨平台部署

## 启动方式
### 本地开发启动
uvicorn main:app --reload

### Docker 一键启动
docker-compose up -d
